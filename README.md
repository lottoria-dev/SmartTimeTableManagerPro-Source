# 🗓️ 스마트 시간표 매니저 Pro (Smart Timetable Manager Pro)
학교 현장의 복잡한 시간표 변동(결강, 보강, 교환 등)을 손쉽게 해결하기 위해 개발된 윈도우용 데스크톱 애플리케이션입니다. 클릭 몇 번으로 복잡한 얽힘을 풀고, AI 기반 자동 배치 기능으로 교무 업무의 부담을 획기적으로 줄여줍니다.

## ✨ 주요 기능
👁️ 다양한 맞춤형 뷰어: 전체 주간 시간표, 요일별, 학급별, 교사별, 교과별로 필터링하여 확인 가능.

🔄 직관적인 맞교환: 두 수업을 클릭하여 즉시 교환 (충돌 및 이동 가능 여부 색상 표시).

🛠️ 손쉬운 보강 배정: 결강 발생 시 해당 시간에 비어있는 교사 목록을 자동 추천하여 즉시 배정.

🔗 연쇄 이동 및 AI 자동 탐색: 빈 공간이나 다른 수업을 밀어내며 이동하는 연쇄 모드 및, AI가 징검다리 이동 경로를 찾아주는 자동 모드 지원.

📋 완벽한 클립보드 호환: 작업 결과를 복사하여 엑셀(Excel)이나 한글(HWP) 문서에 서식 깨짐 없이 바로 붙여넣기.

🛡️ 안전 장치: 실수 방지를 위한 셀 우클릭 잠금 기능, 교사 3연강 시 보라색 경고 표시, 무제한 실행 취소(Undo) 지원.

## 🚀 다운로드 및 실행 방법
이 프로그램은 설치가 필요 없는 단일 실행 파일(.exe) 형태로 제공됩니다.

- Releases 페이지로 이동합니다.
- 최신 버전의 SmartTimetableManagerPro.exe 파일을 다운로드합니다.
- 기초 시간표 작성을 위한 .xlsx 파일도 함께 다운로드합니다.
- 다운로드한 exe 파일을 더블 클릭하여 실행합니다.

## 📝 데이터 준비 및 사용법
프로그램을 사용하기 위해서는 학교의 기초 시간표 데이터가 필요합니다.

1. 양식 작성
- 다운로드한 기본 양식을 엑셀에서 엽니다.
- 기존 나이스(NEIS)나 다른 프로그램의 데이터를 가공하여 붙여넣습니다..

2. 파일 저장
- 작성이 완료되면 엑셀에서 [파일] > [다른 이름으로 저장]을 클릭합니다.
- 파일 형식을 반드시 CSV UTF-8 (쉼표로 분리) (*.csv)로 선택하여 저장합니다.

3. 프로그램에서 불러오기
- 프로그램 상단의 [📂 불러오기] 버튼을 클릭하여 저장한 CSV 파일을 엽니다.
- 원하는 모드(맞교환, 보강 등)를 선택하여 시간표를 수정하고, [📋 복사] 버튼을 눌러 업무에 활용하세요.
